**Title:** BlockTermState
**Summary:** The BlockTermState class holds all the state required for the PostingsReaderBase to produce a PostingsEnum without re-seeking the terms dict. It has a constructor that initializes the state, a method copyFrom that copies the state from another BlockTermState object, and a method toString that returns a string representation of the state. The core logic of the class is to hold the state required for PostingsReaderBase to produce a PostingsEnum without re-seeking the terms dict. The inputs are the TermState object to copy from, and the outputs are the state variables docFreq, totalTermFreq, termBlockOrd, and blockFilePointer. The class extends the OrdTermState class and imports the necessary classes from the Lucene library.
