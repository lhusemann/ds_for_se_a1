**Title:** Live/Deleted Documents Format
**Summary:** This module provides a format for live/deleted documents. It includes methods for reading and writing live docs bits, as well as recording all files in use by a segment commit info into a collection of files. The core logic involves reading and writing live docs bits, as well as recording files in use by a segment commit info. The module takes in a directory, segment commit info, and IO context as inputs for reading live docs. It also takes in bits, directory, segment commit info, new del count, and IO context as inputs for writing live docs. The module does not have any specific outputs. It depends on the Apache Lucene library, specifically the Bits, Directory, IOContext, SegmentCommitInfo, and IOException classes.
