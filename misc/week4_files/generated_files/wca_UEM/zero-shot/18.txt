The KnnVectorsFormat class is used for encoding and decoding per-document vectors and associated indexing structures required for nearest-neighbor search. It provides methods for writing and reading vectors to and from an index, as well as searching for nearest-neighbor vectors. The class is abstract and requires implementation of the fieldsWriter and fieldsReader methods, which are responsible for writing and reading vectors to and from the index. The EMPTY format is provided as a sentinel indicating a codec that does not support vectors. The code also includes dependencies on other classes and interfaces, such as SegmentReadState, SegmentWriteState, VectorValues, TopDocs, TopDocsCollector, Bits, and NamedSPILoader.
