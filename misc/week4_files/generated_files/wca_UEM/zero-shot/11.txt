This code implements a custom FST outputs implementation that stores block data, long ordStart, and long numTerms. The FSTOrdsOutputs class extends the Outputs class and overrides various methods to handle the custom output type. The common method calculates the common prefix between two outputs, the subtract method removes a prefix from an output, the add method adds a prefix to an output, the write method writes an output to a DataOutput, the read method reads an output from a DataInput, the skipOutput method skips an output in a DataInput, the skipFinalOutput method skips the final output in a DataInput, the getNoOutput method returns the no output instance, the outputToString method converts an output to a string, and the newOutput method creates a new output instance. The ramBytesUsed method calculates the RAM bytes used by an output.
