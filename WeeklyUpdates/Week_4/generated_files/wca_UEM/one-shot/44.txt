**Title:** LuceneCodecWriter — Lucene Codec Implementation for Writing Stored Fields

**Summary:**
LuceneCodecWriter is a module that implements the Lucene Codec API for writing stored fields. It is designed to be used in a write-only context, where documents are iteratively added and the finish() method is called after all documents have been written. The finish() method performs verification and sanity-checks. The writer is then closed.

The module follows a clear separation of concerns, with the core logic residing in the `write()` method. It iterates over each document and its fields, writing them to the output in the appropriate format. The finish() method verifies the written data and ensures that the expected number of documents and fields are present.

The module is designed to be used in a write-only context and does not provide any means of reading or seeking within the written data. It isolates the writing logic from the rest of the application, allowing for a clear and focused API.
