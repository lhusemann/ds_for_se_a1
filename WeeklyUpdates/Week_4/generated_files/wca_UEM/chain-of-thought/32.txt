**Title:** PointsReader API for Lucene's PointValues
**Summary:** The PointsReader class is an abstract API for visiting point values. It provides methods for checking the integrity of the reader, getting PointValues for a given field, and getting an instance optimized for merging. The core logic of the class involves checking the integrity of the reader and getting PointValues for a given field. The checkIntegrity method does not take any inputs and does not return any outputs. The getValues method takes a field name as input and returns a PointValues object. The getMergeInstance method does not take any inputs and returns an instance of PointsReader. The PointsReader class imports the Closeable and IOException classes from the java.io package. It also imports the PointValues class from the org.apache.lucene.index package.
