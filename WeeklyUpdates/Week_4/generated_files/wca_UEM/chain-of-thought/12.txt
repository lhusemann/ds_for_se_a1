**Title:** FST Outputs for FSTTermsWriter
**Summary:** This module provides an implementation of the FST Outputs class for the FSTTermsWriter. It includes a TermData class that represents the metadata for one term, and methods for adding, subtracting, and writing TermData objects to a DataOutput. The FSTTermOutputs class is used to define the outputs for the FSTTermsWriter, and it takes a FieldInfo object as a parameter in its constructor to determine whether the field has position information. The common method compares two TermData objects and returns the smaller one if they are comparable, while the subtract method subtracts one TermData object from another and returns the result. The add method adds two TermData objects and returns the result. The write method writes a TermData object to a DataOutput, and the read method reads a TermData object from a DataInput. The skipOutput method skips over the output in a DataInput. The statsEqual and bytesEqual methods are used to compare the doc frequency, total term frequency, and bytes of two TermData objects. The outputToString method returns a string representation of a TermData object.
