**Title:** TermStats Class for Per-Term Statistics
**Summary:** The TermStats class holds per-term statistics, with fields for docFreq (number of documents with at least one occurrence of the term) and totalTermFreq (total number of times the term occurs across all documents in the field). The class has a constructor that takes in the docFreq and totalTermFreq as parameters and initializes the fields. The core logic of this code is to define a class that holds per-term statistics. The TermStats class does not contain any complex logic or data transformations. The class has two fields, docFreq and totalTermFreq, which are initialized in the constructor. The constructor takes in the docFreq and totalTermFreq as parameters. The TermStats class does not depend on any external libraries or dependencies.
