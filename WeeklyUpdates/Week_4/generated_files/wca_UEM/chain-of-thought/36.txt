**Title:** Lucene90PostingsWriterBase
**Summary:** This class is a base for writing postings in a Lucene index. It is part of the Lucene project's codecs module and is used by the Lucene90BlockTreeTermsWriter class to handle writing postings. The class implements the Closeable interface to allow for resource cleanup. It provides methods for initializing the writer, writing terms, encoding term metadata, and setting the current field for writing. The writeTerm method takes a term, a TermsEnum to iterate over postings, a FixedBitSet to track which documents have been seen, and a NormsProducer to write norms. It returns a BlockTermState object that contains metadata about the term's postings. The encodeTerm method encodes term metadata as long[] and byte[], and the setField method sets the current field for writing. The class also provides a constructor to allow for subclassing.
