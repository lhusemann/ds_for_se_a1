**Title:** Lucene 9.0 Postings Writer
**Summary:** This module is a Java class that implements a postings writer for Lucene 9.0. It is used to write docId(maybe frq,pos,offset,payloads) list with postings format. The postings list for each term is stored separately. The module includes methods for initializing the postings writer, starting a new term, starting a new document, adding a position to the document, finishing a document, finishing a term, encoding a term, and closing the postings writer. It also includes variables and imports for handling docId, frequency, position, offset, payload, and norms.
