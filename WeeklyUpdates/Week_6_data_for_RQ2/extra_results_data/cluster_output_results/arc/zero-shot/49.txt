This code defines the abstract class PostingsReaderBase, which is used by the core terms dictionaries to manage the creation of PostingsEnum and ImpactsEnum instances. It provides an IndexInput where this class may read any previously stored data that it had written in its corresponding PostingsWriterBase at indexing time. The class includes methods for initializing the terms dictionary, decoding metadata for the next term, creating empty TermState, and checking the integrity of the reader.
