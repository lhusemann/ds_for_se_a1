This code provides a custom PostingsFormat implementation for Lucene that uses the OrdsBlockTreeTermsWriter and Lucene90PostingsWriter. It allows for the configuration of the minimum and maximum block sizes for the block terms dictionary. The BLOCK_SIZE constant defines the fixed packed block size, which must be a multiple of 64. The code also includes methods for creating the OrdsBlockTreeTermsWriter and OrdsBlockTreeTermsReader, as well as handling the success and failure of these operations.
