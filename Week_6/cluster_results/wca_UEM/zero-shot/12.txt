This code implements an FST (Finite State Transducer) Outputs class for the FSTTermsWriter. The FSTTermOutputs class handles the metadata for each term in the FST. The TermData class represents the metadata for one term and implements the Accountable interface to estimate the memory usage of the TermData object. The FSTTermOutputs class overrides several methods from the Outputs class, including common, subtract, add, write, read, skipOutput, getNoOutput, and outputToString. The common method compares two TermData objects and returns the smaller one, while the subtract method subtracts one TermData object from another. The add method adds two TermData objects and returns the result. The write method writes the TermData object to a DataOutput stream, and the read method reads a TermData object from a DataInput stream. The skipOutput method skips over the TermData object in a DataInput stream. The getNoOutput method returns the NO_OUTPUT object, and the outputToString method converts a TermData object to a string representation. The statsEqual and bytesEqual methods are used to compare the docFreq and totalTermFreq values, as well as the bytes array, of two TermData objects.
