The MutablePointValues class is an internal class used by codecs to optimize flush operations. It extends the PointValues class and provides abstract methods to manipulate the order of points. The class includes methods to get the packed bytes of a value, get a specific byte of a value, get the doc ID of a value, swap two values, save a value to temporary storage, and restore values from temporary storage. These methods allow codecs to efficiently manipulate the order of points and optimize flush operations.
