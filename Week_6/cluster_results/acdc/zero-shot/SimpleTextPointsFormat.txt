Title: Lucene's SimpleTextPointsFormat Reader and Abstract API for Writing Points

Description:
This code provides a reader for Lucene's SimpleTextPointsFormat, which reads data from a file and provides access to the point values stored in the file. The SimpleTextPointsReader class initializes readers for each field in the index and provides methods to retrieve the point values for a given field. The SimpleTextBKDReader class is used to read the actual point values from the file. The SimpleTextPointsReader class also provides methods to check the integrity of the file and to close the reader.

Additionally, this code provides an abstract API for writing points in Lucene. It includes a PointsWriter class with methods for writing all values from a provided reader, as well as a default merge implementation for one field. The merge method merges incoming points readers by visiting all their points and adding to this writer. The finish method is called once at the end before close.

The PointsReader class is used to visit point values. It implements the Closeable interface and provides several methods for accessing and checking the integrity of point values. The class is part of the Lucene library and is marked as experimental, meaning it may undergo changes or be removed in future versions.

The PointsFormat class is used to encode and decode indexed points. It provides methods for creating a new point format, writing a new segment, and reading a segment. The EMPTY field is a PointsFormat that has nothing indexed.
