This code is a writer for Lucene's stored fields. It writes plain-text stored fields in a Lucene index. The SimpleTextStoredFieldsWriter class extends the StoredFieldsWriter class and overrides its methods to write the fields to a text file. The startDocument() method writes the document number to the output stream. The writeField() method writes the field name, type, and value to the output stream. The finish() method writes an end marker to the output stream and writes a checksum. The close() method closes the output stream. The SimpleTextStoredFieldsWriter class also includes static final BytesRef constants for different field types, such as string, binary, int, long, float, and double. These constants are used to write the field type to the output stream. The SimpleTextStoredFieldsWriter class also includes a scratch BytesRefBuilder for building BytesRef objects. The SimpleTextStoredFieldsWriter class is marked as experimental, meaning it is not intended for production use.
