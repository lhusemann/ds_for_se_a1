FSTPostingsFormat implements a custom postings format using an FST (Finite State Transducer) term dictionary and the Lucene50PBF (Lucene 5.0 Postings Format). BlockTreeOrdsPostingsFormat is a PostingsFormat implementation that provides customizable settings for the block terms dictionary. It allows the user to specify the minimum and maximum block sizes for the terms dictionary.
