This code provides an abstract API to write points in Lucene. It includes a PointsWriter class that implements the abstract API. The PointsWriter class has methods to write all values contained in a provided reader, as well as a default merge implementation for one field. The merge implementation re-indexes all the values from the incoming segment. The PointsWriter class also has a merge method that merges incoming points readers by visiting all their points and adding to this writer. Finally, the PointsWriter class has a finish method called once at the end before close.
