# Information of folder

## cluster_output_results
This folder contains descriptive summaries of the cluster created by ACDC, ARC, LIMBO and WCA Algorithms from `file-based-rsf/` created with a prompt that complies with changes made on 2026-06-02. They were created by the files in `Extra_hierarchical_summarization/`.

## Extra_hierarchical_summarization
This folder contains the scripts used for descriptive summaries that comply with the changes made on 2026-06-02.

## file-based-rsf
This folder contains the clusters generated by the algorithms from previous weeks in file-format.

## formated_results
This folder contains the formated results of `cluster_output_results` to comply the new format announced on 2026-06-02. It also contains the script used to format them. Although the titles were added manually as the LLM results don't contain titles for all clusters.
