**Title:** KnnVectorsWriter: A Writer for Vectors in an Index
**Summary:**
The KnnVectorsWriter class is a writer for vectors in an index. It implements the Closeable interface and has a single constructor. The writeField method takes a FieldInfo object and a VectorValues object as parameters and writes all the values contained in the VectorValues object to the index. The finish method is called once at the end before the close method is called. The merge method is used to merge the vector values from multiple segments for all fields. It takes a MergeState object as a parameter and iterates over the fieldInfos in the MergeState object. For each fieldInfo, it checks if it has vector values and then calls the mergeVectors method. The mergeVectors method iterates over the knnVectorsReaders in the MergeState object and creates a new VectorValues object by iterating over the sub vectors, mapping the resulting doc ids using the docMaps in the MergeState object. Finally, it calls the writeField method with the MergeFieldInfo and the new VectorValues object.
