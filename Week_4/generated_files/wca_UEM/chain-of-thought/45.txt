**Title:** TermStats Class
**Summary:**
The TermStats class is a container for per-term statistics. It holds two fields: docFreq, which represents the number of documents that have at least one occurrence of the term, and totalTermFreq, which represents the total number of times the term occurs across all documents in the field. The class has a single constructor that takes in the docFreq and totalTermFreq as parameters and assigns them to the corresponding fields. The TermStats class does not have any external dependencies and only uses the built-in Java types int and long.
