**Title:** Base class for hashing functions that can be referenced by name
**Summary:** This code provides a base class for hashing functions that can be referenced by name. It is expected that subclasses will provide thread-safe implementations of the hash function on the range of bytes referenced in the provided BytesRef. The core logic of this code is to provide a base class for hashing functions that can be referenced by name. It is expected that subclasses will provide thread-safe implementations of the hash function on the range of bytes referenced in the provided BytesRef. The code takes a BytesRef as input and returns the hash of the bytes referenced by bytes.offset and length bytes.length. This code depends on the org.apache.lucene.util.BytesRef class.
