**Title:** PostingsFormat
**Summary:** The PostingsFormat class is a Java representation of a PostingsFormat. It is used to encode and decode terms, postings, and proximity data. The class implements the NamedSPILoader.NamedSPI interface and provides methods for writing a new segment and reading a segment. The class also provides methods for looking up a format by name, getting a list of all available format names, and reloading the postings format list from a given ClassLoader. The class uses Java's ServiceLoader Service Provider Interface (SPI) to resolve format names. The writing of a new segment involves creating a FieldsConsumer, which encodes the terms, postings, and proximity data. The reading of a segment involves creating a FieldsProducer, which decodes the terms, postings, and proximity data.
