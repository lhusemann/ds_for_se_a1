**Title:** MutablePointValues: A Java Class for Optimizing Flushing in Codecs

**Summary:**
The MutablePointValues class is a Java implementation used by codecs to optimize flushing by allowing the order of points to be changed. It extends the PointValues class and provides methods for accessing and manipulating the values, including getting the packed bytes of a value, getting the k-th byte of a value, getting the doc ID of a value, swapping two values, saving a value to temporary storage, and restoring values from temporary storage. The class is part of the Lucene library and is used for codecs that require efficient data manipulation.
