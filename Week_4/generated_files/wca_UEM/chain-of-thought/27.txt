**Title:** MutablePointValues: Optimizing Flushing in Lucene Codecs
**Summary:**
The MutablePointValues class is an abstract extension of the PointValues class in Lucene codecs. It provides methods for accessing and manipulating point values, including getting the packed bytes of a value, getting the k-th byte of a value, getting the doc ID of a value, swapping two values, saving a value to temporary storage, and restoring values from temporary storage. The core logic of the class involves providing methods for accessing and manipulating point values. The getValue method retrieves the packed bytes of a value, while the getByteAt method retrieves the k-th byte of a value. The getDocID method returns the doc ID of a value, and the swap method swaps the positions of two values. The save method saves a value to temporary storage, and the restore method restores values from temporary storage. The class has no external dependencies.
