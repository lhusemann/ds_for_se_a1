**Title:** Lucene DocValues Format
**Summary:** This module is an abstract class that defines the functionality of encoding and decoding per-document values. It provides a way to retrieve a specific format by name using Java's Service Provider Interface (SPI). The class includes a constructor that takes a name parameter and registers the format with the SPI, a method to create a DocValuesConsumer for writing docvalues to the index, a method to create a DocValuesProducer for reading docvalues from the index, a method to retrieve a format by name using the SPI, a method to retrieve a list of all available format names using the SPI, and a method to reload the list of available formats from a given ClassLoader. The code is part of the Lucene project and is marked as experimental, indicating that it may be subject to change or removal in future versions.
