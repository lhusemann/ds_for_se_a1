**Title:** SkipListWriter
**Summary:** This class is an abstract class that writes skip lists with multiple levels. Skip lists are used to efficiently navigate through posting lists in Lucene, a search engine library. The skip lists contain entries at different levels, with each level containing every skipInterval-th entry from the previous level. The number of entries on each level is calculated based on the document frequency and the skip interval. The class provides methods for writing skip data at different levels, buffering skip data, and writing the buffered skip lists to an output. The skip data is encoded in a subclass-specific format. The class also provides methods for writing the length of a level and the child pointer of a block to an output.
