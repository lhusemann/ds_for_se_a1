**Title:** PointsFormat: A Lucene Points Format for Encoding and Decoding Indexed Points
**Summary:** The PointsFormat class in the org.apache.lucene.codecs package provides functionality for encoding and decoding indexed points. It includes methods for writing a new segment and reading a segment. The class has no superclass and implements no specific interfaces. The fieldsWriter method takes a SegmentWriteState object as input and returns a PointsWriter object. The fieldsReader method takes a SegmentReadState object as input and returns a PointsReader object. The code depends on the org.apache.lucene.index.PointValues, org.apache.lucene.index.SegmentReadState, and org.apache.lucene.index.SegmentWriteState classes. The EMPTY static instance of PointsFormat throws an UnsupportedOperationException when attempting to write a new segment. The fieldsReader method returns a PointsReader instance that throws an IllegalArgumentException if a field is not indexed with points.
