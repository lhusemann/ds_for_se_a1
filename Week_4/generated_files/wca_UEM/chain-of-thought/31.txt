**Title:** PointsFormat: Encoding and decoding indexed points
**Summary:**
The `PointsFormat` class is used to encode and decode indexed points. It provides methods to create a new point format and write a new segment. The core logic of the code is implemented in the `PointsFormat` class. The `PointsFormat` class has the following inputs and outputs:

- Inputs:
  - `SegmentWriteState`: A class that represents the write state of a segment. It contains information about the segment being written, such as the directory, segment name, and other metadata.
  - `SegmentReadState`: A class that represents the read state of a segment. It contains information about the segment being read, such as the directory, segment name, and other metadata.

- Outputs:
  - `PointsWriter`: An interface that defines methods to write points to a segment.
  - `PointsReader`: An interface that defines methods to read points from a segment.

The `PointsFormat` class depends on the following classes and interfaces:

- `org.apache.lucene.index.PointValues`: An interface that defines methods to access point values.
- `org.apache.lucene.index.SegmentReadState`: A class that represents the read state of a segment.
- `org.apache.lucene.index.SegmentWriteState`: A class that represents the write state of a segment.
- `java.io.IOException`: An exception that is thrown when an I/O error occurs.
