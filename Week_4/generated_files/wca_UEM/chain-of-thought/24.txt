**Title:** Lucene90PostingsWriter: A Class for Writing DocId, Position, and Payload Data with Postings Format
**Summary:**
The Lucene90PostingsWriter class is responsible for writing docId(maybe frq,pos,offset,payloads) list with postings format. It extends the PushPostingsWriterBase class and has various methods and variables to handle the writing of postings list for each term. The class initializes the necessary outputs for writing docId, position, and payload data. It provides methods to start a new term and start a new document within that term. It handles the addition of positions, payloads, and offsets to the postings list. It encodes the term state and writes it to an output stream. Finally, it closes the outputs after writing is complete. The core logic of the class involves encoding and writing docId, position, and payload data to the output streams. It also handles the buffering of data and the writing of skip data for each block of docs. The class takes the following inputs: SegmentWriteState, FieldInfo, and NumericDocValues. It produces the following outputs: IndexOutput and IntBlockTermState. The class has the following dependencies: Lucene90SkipWriter, PForUtil, ForUtil, Lucene90PostingsFormat, ArrayUtil, BitUtil, BytesRef, CorruptIndexException, IndexFileNames, IndexOutput, IndexWriter, IOUtils, NumericDocValues, SegmentWriteState, DataOutput, and IOException.
