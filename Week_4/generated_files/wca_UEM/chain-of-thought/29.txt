**Title:** Lucene Norms Format
**Summary:** This module defines a class called NormsFormat that is used to encode and decode per-document score normalization values. The core logic involves the use of abstract classes and methods to define the behavior of encoding and decoding norms. The NormsConsumer class is used to write norms to the index, while the NormsProducer class is used to read norms from the index. The module has two abstract methods, normsConsumer and normsProducer, which take in SegmentWriteState and SegmentReadState objects as inputs, respectively. The normsConsumer method returns a NormsConsumer object, while the normsProducer method returns a NormsProducer object. This code imports several classes from the org.apache.lucene.index and org.apache.lucene.codecs packages, which are part of the Lucene library.
