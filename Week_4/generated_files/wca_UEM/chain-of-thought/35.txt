**Title:** PostingsReaderBase: Manages the creation of PostingsEnum and ImpactsEnum instances
**Summary:** The PostingsReaderBase class is used by the core terms dictionaries to manage the creation of PostingsEnum and ImpactsEnum instances. It provides an IndexInput where the class may read any previously stored data that it had written in its corresponding PostingsWriterBase at indexing time. The class initializes the terms dictionary IndexInput and segment read state, creates a new empty TermState, decodes metadata for the next term, provides a PostingsEnum that fully consumes the state and can be reused, returns an ImpactsEnum that computes impacts with a scorer, and checks the consistency of the reader. The class is marked as lucene.experimental.
