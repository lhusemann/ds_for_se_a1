**Title:** Custom FST Outputs Interface Implementation in Lucene
**Summary:** This module provides a custom implementation of the FST outputs interface in Lucene. It defines a class called FSTOrdsOutputs that extends the Outputs class. The FSTOrdsOutputs class is used to store block data, start ordinal, and end ordinal. It provides methods for adding, subtracting, and writing outputs, as well as calculating the RAM usage of an output. The Output class represents an output with bytes, start ordinal, and end ordinal. It has methods for comparing outputs, calculating hash codes, and returning a string representation. The FSTOrdsOutputs class overrides the common, subtract, add, write, read, skipOutput, skipFinalOutput, getNoOutput, outputToString, and ramBytesUsed methods from the Outputs class. The newOutput method creates a new output with the given bytes, start ordinal, and end ordinal.
