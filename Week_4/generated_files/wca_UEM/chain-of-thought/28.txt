**Title:** NormsConsumer: Abstract Class for Consuming Normalization Values
**Summary:** The NormsConsumer class is an abstract class used to consume normalization values. It provides a method addNormsField that is called for each field with normalization values. The implementation is free to iterate over the values multiple times. After all fields are added, the consumer is closed. The class also has a merge method that merges in the fields from the readers in mergeState. The default implementation calls mergeNormsField for each field, filling segments with missing norms for the field with zeros. Implementations can override this method for more sophisticated merging. The mergeNormsField method merges the norms from toMerge. The default implementation calls addNormsField, passing an Iterable that merges and filters deleted documents on the fly.
