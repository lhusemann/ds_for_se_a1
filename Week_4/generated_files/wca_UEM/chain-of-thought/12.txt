**Title:** FST Outputs for FSTTermsWriter
**Summary:** This module provides functionality for handling term data and metadata in an FST (Finite State Transducer). It includes the TermData class, which represents the metadata for one term, and the FSTTermOutputs class, which implements the common, subtract, add, write, read, skipOutput, getNoOutput, and outputToString methods from the Outputs class. The common method compares two TermData objects and returns the smaller one if they are comparable, while the subtract method subtracts one TermData object from another and returns the result. The add method adds two TermData objects and returns the result. The write method writes the TermData object to a DataOutput, the read method reads a TermData object from a DataInput, the skipOutput method skips over the output in a DataInput, the getNoOutput method returns the NO_OUTPUT TermData object, and the outputToString method converts a TermData object to a string. The statsEqual and bytesEqual methods compare the stats and bytes of two TermData objects, respectively.
