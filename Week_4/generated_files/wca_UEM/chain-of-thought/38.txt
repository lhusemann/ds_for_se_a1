**Title:** Custom Lucene DocValuesFormat for Simple Text Format
**Summary:** This code provides a custom implementation of the Lucene DocValuesFormat class for reading and writing doc values in a simple text format. It overrides the fieldsConsumer and fieldsProducer methods to provide custom implementations for writing and reading doc values. The SimpleTextDocValuesWriter class is responsible for writing doc values to a simple text file, while the SimpleTextDocValuesReader class is responsible for reading doc values from a simple text file. The code depends on the Lucene library and uses the SimpleTextDocValuesReader and SimpleTextDocValuesWriter classes for reading and writing simple text doc values.
