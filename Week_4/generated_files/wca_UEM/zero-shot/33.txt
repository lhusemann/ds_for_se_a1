The code provides an abstract API to write points, with a class called PointsWriter that implements the abstract API. The PointsWriter class has a method called writeField that writes all values contained in the provided reader. It also has a method called mergeOneField that is a default naive merge implementation for one field. The merge method is a default merge implementation to merge incoming points readers by visiting all their points and adding to this writer. The finish method is called once at the end before close.
