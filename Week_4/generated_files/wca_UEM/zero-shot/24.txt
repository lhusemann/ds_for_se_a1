This code provides a concrete implementation for writing postings list with postings format for Lucene 9.0. It is a class named Lucene90PostingsWriter that extends the PushPostingsWriterBase class. The class is responsible for writing postings list for each term separately. It has various member variables and methods for writing postings, managing skip data, and encoding term information. The code also includes various member variables for managing postings data, skip data, and term state.
