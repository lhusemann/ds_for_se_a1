This code provides an abstract API to visit point values. It includes a class called `PointsReader` that implements the `Closeable` interface. The class has a constructor that is called by subclass constructors. It also includes a method called `checkIntegrity` that checks the consistency of the reader. The `getValues` method returns `PointValues` for a given field. Finally, the `getMergeInstance` method returns an instance optimized for merging.
