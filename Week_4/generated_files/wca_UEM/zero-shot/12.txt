This code implements a custom FST Outputs implementation for the FSTTermsWriter. The FSTTermOutputs class overrides the methods of the Outputs class to provide custom functionality for handling term data. The TermData class represents the metadata for one term and implements the Accountable interface to provide information about the memory usage of the term data. The common, subtract, add, write, read, skipOutput, getNoOutput, and outputToString methods are overridden to provide custom logic for handling term data. Static helper methods statsEqual and bytesEqual are used to compare the statistics and bytes of two TermData objects. Overall, this code provides an efficient and customizable implementation of the Outputs interface for use in the FSTTermsWriter.
