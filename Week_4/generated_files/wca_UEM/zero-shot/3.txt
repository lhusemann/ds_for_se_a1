This code provides a base for encoding and decoding an inverted index segment. It uses the Service Provider Interface (SPI) to resolve codec names and provides methods for encoding/decoding postings, docvalues, stored fields, term vectors, field infos, segment info, document normalization values, live docs, compound files, points index, and numeric vector fields. It also provides methods for looking up a codec by name, getting a list of all available codec names, reloading the codec list from a given classloader, and setting the default codec used for newly created IndexWriterConfig objects.
