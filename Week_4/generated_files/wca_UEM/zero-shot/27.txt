The `MutablePointValues` class is a Lucene codec implementation that extends the `PointValues` class. It provides methods to modify the order of points during flushing, which can be useful for optimization purposes. The class includes methods to retrieve the packed bytes of a value, retrieve the k-th byte of a value, retrieve the doc ID of a value, swap two values, save a value to temporary storage, and restore values from temporary storage to the original storage.
