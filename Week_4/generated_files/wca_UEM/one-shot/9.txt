**Title:** DocValues Encoding and Decoding Module for Lucene

**Summary:**
This module provides classes and methods for encoding and decoding per-document values in Lucene. It includes a base class `DocValuesFormat` and its implementations, as well as helper classes and methods for loading and managing docvalues formats. The `DocValuesFormat` class provides abstract methods for creating `DocValuesConsumer` and `DocValuesProducer` instances, which are used to write and read docvalues to and from the index, respectively. The `forName` method is used to look up a format by name, and the `reloadDocValuesFormats` method is used to reload the list of available docvalues formats from a given classloader.

The module follows a clear separation of concerns, with the `DocValuesFormat` class acting as a facade for the encoding and decoding logic, and the `DocValuesConsumer` and `DocValuesProducer` interfaces exposing the write and read operations. The `SegmentReadState` and `SegmentWriteState` instances are used to manage the state of the index segment being processed, while the `ClassLoader` is used for loading and managing docvalues formats.

The code is designed to be highly pluggable and extensible, allowing for easy addition of new docvalues formats and customizations. It also follows best practices for Lucene indexing and encoding, ensuring high performance and efficiency.
