**Title:** DocValuesFormat — Per-Document Value Encoding and Decoding

**Summary:**
The `DocValuesFormat` class is a core component of the Lucene library, providing functionality for encoding and decoding per-document values. It is responsible for reading and writing docvalues to and from an index.

The class defines methods for creating a `DocValuesConsumer` to write docvalues to the index and a `DocValuesProducer` to read docvalues from the index. The `DocValuesConsumer` and `DocValuesProducer` classes handle the actual encoding and decoding of docvalues.

The `DocValuesFormat` class is part of the Lucene library and is marked as experimental, meaning it may undergo changes or have limited support.
