**Title:** Norms Format — Per-Document Score Normalization Format

**Summary:**
The NormsFormat class is a low-level module for encoding and decoding per-document score normalization values. It is designed to be pluggable and extensible, allowing custom formats to be plugged in as needed. The abstract NormsFormat class provides the core logic for reading and writing norms, while the NormsConsumer and NormsProducer interfaces define the contract for writing and reading norms, respectively.

The NormsFormat module is intended to be used by higher-level indexing and search components, such as the IndexWriter and IndexReader classes. It is a low-level, narrowly focused component that is not expected to be used directly by most users.

The NormsFormat module is part of the Lucene indexing and search library, which is a widely used open-source search engine library written in Java. It is designed to be fast, scalable, and easy to use, making it a popular choice for search-intensive applications.
