**Title:** CustomFSTOutputs — Custom FST Outputs Implementation for Lucene

**Summary:**
CustomFSTOutputs is a custom implementation of the FST outputs interface for Lucene. It is used to store block data (BytesRef), long ordStart, and long numTerms. It provides custom implementations of common, subtract, add, write, read, skipOutput, skipFinalOutput, getNoOutput, outputToString, and ramBytesUsed methods. These methods handle the manipulation and storage of block data, ordStart, and numTerms.

The code follows a clear separation of concerns, with the core logic implemented in the custom methods. It uses the DataInput and DataOutput streams for reading and writing data, and it depends on the Lucene utility classes BytesRef, RamUsageEstimator, and StringHelper.

CustomFSTOutputs is designed to be used in applications that require custom FST outputs for Lucene, such as text search and indexing. It can be easily integrated into existing Lucene applications by replacing the default FST outputs with this custom implementation.
