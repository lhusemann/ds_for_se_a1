**Title:** FlushablePointsReader — Point Values Reader with Flushing Capabilities

**Summary:**
FlushablePointsReader is a utility class that extends the Lucene PointValues interface to include a flush method. This allows codecs to optimize flushing by changing the order of points. The class provides methods to retrieve packed bytes, individual bytes, and doc IDs of the i-th value, as well as methods to swap values, save values to temporary storage, and restore values from temporary storage.

The class is designed to be used by codecs that require the ability to change the order of points during flushing. It is a low-level utility that is not typically used directly by application code.

FlushablePointsReader is part of the Lucene codecs module, which provides classes for encoding and decoding index data. It is intended for use by Lucene internals and is not expected to be used by applications directly.
