**Title:** PostingsWriter — Lucene 9.0 Postings List Writer

**Summary:**
The PostingsWriter module is a high-performance postings list writer for Lucene 9.0. It is responsible for writing postings lists for each term in a separate file. The module is designed to be efficient and scalable, handling large datasets with ease.

The core logic of the module is implemented in the `writePostings` method, which iterates over the input list and writes the postings lists for each term to separate files. The method uses a separate `PostingsConsumer` instance to handle the writing of each term's postings list.

The module is designed to be modular and extensible, allowing for easy integration with other Lucene components. It depends on the `java.util.List` and `java.util.ArrayList` classes for data storage and manipulation, and it is built to be compatible with the Lucene 9.0 API.
