**Title:** FSTTermOutputs — FST Outputs Implementation for Lucene's FSTTermsWriter

**Summary:**
The FSTTermOutputs class implements an FST Outputs implementation for Lucene's FSTTermsWriter. It provides functionality for writing and reading TermData objects, which represent metadata for each term in the FST. The class includes methods for writing and reading TermData objects to and from a DataOutput and DataInput stream, respectively. It also provides methods for performing common, subtract, and add operations on TermData objects.

The architecture follows a clear separation of concerns, with the FSTTermOutputs class owns the iteration and accumulation logic using a standard ArrayList. Data flows in one direction: the input list is passed down to the filter, elements are tested via the predicate layer, and matching values are collected into a new list that is returned to the caller.

There is no shared mutable state anywhere in the module, making it deterministic and safe for concurrent use. Its only external dependencies are the standard Java Collections API, making it a self-contained leaf cluster with minimal coupling to the broader codebase.
