**Title:** Lucene Live Documents Format

**Summary:**
The `LiveDocsFormat` class provides functionality for reading and writing live documents in Lucene. It defines methods for reading live document bits from a directory and a segment commit info, and returning them as a `Bits` object. It also provides a method for persisting live document bits to a directory, using the next generation of the deletes file determined by the segment commit info. Additionally, it has a method for recording all files in use by a segment commit info into a collection of strings.

The class follows a clear separation of concerns, with the `readLiveDocs` and `writeLiveDocs` methods handling the actual reading and writing of live document bits, while the `files` method isolates the logic for recording files. It is designed to be used as a low-level utility for Lucene, with clear inputs and outputs, and minimal external dependencies.

The class isolates the file format for live documents from the rest of Lucene, making it easier to evolve the format in the future. It is also designed to be pluggable, allowing for different implementations of live documents formats in the future.
