**Title:** SimpleText — Text File Reading and Writing Utilities

**Summary:**
The SimpleText package provides utilities for reading and writing text files in a simple, efficient, and reliable format. It is designed to be used for high-performance data processing tasks.

The package includes a set of utility methods for writing text data to an output stream. These methods convert the input data into bytes and write them to the output stream, escaping special characters as necessary. The SimpleText format supports writing strings, bytes, and newline characters. It also includes methods for writing and checking checksums for the written data.

The package also includes a set of utility methods for reading text data from an input stream. These methods read bytes from the input stream, unescaping special characters as they are encountered. The SimpleText format supports reading lines from the input data.

The SimpleText package is designed to be used in a high-performance context and is optimized for speed and efficiency. It is thread-safe and does not use any shared mutable state, making it safe for concurrent use.

The SimpleText package is a self-contained module with minimal external dependencies. It only uses the standard Java I/O and Lucene APIs, making it easy to integrate into existing projects.
