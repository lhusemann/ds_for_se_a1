**Title:** VersionedHeaderUtil — Utility for Reading and Writing Versioned Headers

**Summary:**
VersionedHeaderUtil is a utility class that provides methods for writing and reading versioned headers. It is designed to be used by Lucene codecs for writing and reading headers with specific formats. The class provides methods for writing codec headers and index headers, as well as checking the validity of these headers.

The core logic of the code involves writing and reading headers with specific formats and performing checksum validation. The class isolates the header writing and reading logic from the rest of the codec, allowing for easy reuse and maintenance.

The inputs to the class include the output stream, codec name, version number, object ID, and suffix. The outputs are the actual version found, or an exception is thrown if the header is invalid. The class depends on the `java.io`, `java.nio`, `java.util`, and `org.apache.lucene` packages.

The code is well-designed and easy to understand, with clear separation of concerns and a focus on reusability and maintainability. It is a valuable utility for codecs that need to write and read versioned headers in a consistent and reliable manner.
