**Title:** Lucene NormsConsumer for Merging Normalization Values

**Summary:**
The Lucene NormsConsumer is a module that provides functionality for merging normalization values. It defines a class `NormsConsumer` that implements the `Closeable` interface. The `NormsConsumer` class is responsible for merging normalization values for a field from multiple readers.

The `NormsConsumer` class has two key methods: `mergeNormsField` and `merge`. The `mergeNormsField` method merges and filters deleted documents on the fly. The `merge` method iterates over each field with normalization values and calls the `mergeNormsField` method.

The `NormsConsumer` class has no specific inputs or outputs. It depends on the Lucene library and its dependencies.

The `NormsConsumer` class follows a standard design pattern for Lucene modules, providing a clear separation of concerns and reusability across different parts of the library.
