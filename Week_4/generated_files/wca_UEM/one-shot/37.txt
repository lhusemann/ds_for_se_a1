**Title:** SegmentInfoFormat — Controls the Format of SegmentInfo in Apache Lucene

**Summary:**
The SegmentInfoFormat class is responsible for reading and writing the SegmentInfo (segment metadata file) in Apache Lucene. It is a low-level utility class that provides the core logic for reading and writing SegmentInfo data.

The `read` method is used to read SegmentInfo data from a directory. It takes a `Directory` object, the name of the segment to read, an expected identifier for the segment, and an `IOContext` object. It returns a `SegmentInfo` object populated with the read data.

The `write` method is used to write SegmentInfo data. It takes a `Directory` object, a `SegmentInfo` object, and an `IOContext` object. It writes the data to the directory and adds the SegmentInfo filename(s) to the `info` object.

The class has no shared mutable state and is safe for concurrent use. It depends only on the standard Java Collections API, making it a self-contained leaf cluster with minimal coupling to the broader codebase.
