**Title:** TermVectorsFormat — Controls the Format of Term Vectors

**Summary:**
TermVectorsFormat is a low-level format for encoding and decoding term vectors. It is designed to be pluggable, allowing for custom implementations to be dropped in as necessary. It provides two key methods for each term vector: `TermVectorsReader` for reading and `TermVectorsWriter` for writing.

The TermVectorsFormat API is intended to be used by higher-level formats like theLucene segment format, which use the specific encodings provided by the defaultLucene TermVectorsFormat implementation. TermVectorsFormat is typically only used by custom formats, such as for custom scoring or for specialized retrieval needs.
